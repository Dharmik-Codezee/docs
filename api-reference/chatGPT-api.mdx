## Endpoint

`POST /api/text/auto-generate`

This endpoint automatically generates text based on provided data.

### Request

- Headers
  - `authorization`: Authorization token for access.
- Body Parameters
  - `msg_lang` (Optional): Language code for the generated text (defaults to 'en').
  - `business_id` (Optional): ID of the business.
  - `experience`: Experience level required for the job.
  - `job_title`: Title of the job position.
  - `salary`: Salary offered for the job.
  - `location`: Location of the job.
  - `specialization`: Specialization required for the job.
  - `job_type`: Type of job (e.g., full-time, part-time).
  - `workplace_type`: Type of workplace (e.g., office, remote).

### Response

- Status: 
  - 200 OK if the operation is successful.
  - Error status codes for failures.
- Body: JSON response containing the generated text or error message.
  - If the text generation is successful, the generated text is returned.
  - If the text generation fails, an error message is returned.

### Notes

- This endpoint requires authentication via the `authorization` header.
- The request body should include parameters necessary for text generation.
- Error messages are returned for various scenarios such as user not existing, business not existing, user not being a member of the business, etc.


### Request Body

```json
{
    "business_id": "65bcb3c9b9b448f7b6d52925",
    "job_title":"Manager",
    "specialization":"Sales & Customers services",
    "job_type": "Full time",
    "experience": "Fresher",
    "salary": "50000",
    "location": "Bangkok",
    "workplace_type": "On site"
}
```
