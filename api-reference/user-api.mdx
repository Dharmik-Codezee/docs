# User API Documentation

## Endpoint: /api/user/signup/:user_type

This endpoint is used to create a new user.

### Request

```json
{
    "first_name": "Somchai",  
    "last_name": "Anant",
    "country_code": "66",
    "mobile_no": "655-597-453",
    "user_email": "somchai@gmail.com",
    "date_of_birth":"2024-01-18",
    "gender": "male",
    "profile_img": "images/photo.jpeg",
    "identity_card":"images/id_card.jpeg",
    "skills": ["leadership", "communication", "marketing"],
    "notification": true,
    "open_to_work": true,
    "language": 1
}
```
