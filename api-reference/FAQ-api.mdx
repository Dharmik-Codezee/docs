# List FAQ API

## Endpoint

`GET /api/faq/list`

This endpoint retrieves a list of frequently asked questions (FAQs) from the help center.

### Request

- Headers
  - `authorization`: Authorization token for the authenticated user.

### Response

- Status:
  - 200 OK if the operation is successful.
  - Error status codes for failures.

- Body: JSON response containing the list of FAQs.
  - If FAQs are found, they are returned in the response.
  - If no FAQs are available, an empty array is returned.

### Notes

- Authentication is required via the `authorization` header.
- The API fetches FAQs from the help center database.
- The response includes a list of FAQs if available.
- Error messages are returned for scenarios such as unauthorized access or internal server errors.
